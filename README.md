## How to Run The Project

### Requirements
1. Node.js (Front-end)
2. MySql
3. VS Code
4. Angular CLI
5. Bun.js (Back-end) [Download](https://bun.com/docs/installation#windows)

### To Run Server
1. Make sure you have a running MySql Server so that the server can connect with the db

2. Run the command
```bash
cd server
npm install
npm run dev
```

### To Run Client
```bash
cd client
npm install
ng serve --open
```
