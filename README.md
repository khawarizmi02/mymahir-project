## How to Run The Project

### Requirements
1. Node
2. MySql
3. VS Code
4. Angular CLI

### To Run Server
1. Make sure you have a running MySql Server so that the server can connect with the db

2. Run the command
```bash
cd server
npm run dev
```

### To Run Client
```bash
cd client
ng serve --open
```
